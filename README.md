# Artifacts Website

This is a web application for an artifacts website that displays a diverse collection of items including coins, war collections, paintings, furniture, and clothing. The website provides an immersive experience for users to explore and appreciate these artifacts. 


## User Details and MongoDB

The website utilizes MongoDB to store user details securely. When users sign up or log in to the artifacts website, their information such as username, email, and encrypted password is stored in the MongoDB database. This allows for user authentication and personalized experiences.

## Technologies Used

- HTML, CSS, and JavaScript for the frontend development
- MongoDB for storing user details
- Backend technologies such as Node.js and Express.js for server-side logic
